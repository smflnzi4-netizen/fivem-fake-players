# FiveBoosts API Wrapper & SDK (Official)

[![License: MIT](https://img.shields.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.shields.io/badge/FiveM-v2.0_Compatible-blue)](https://fiveboosts.net)

This repository provides an easy-to-use API wrapper for connecting your automation tools, Discord bots, and CI/CD deployment pipelines directly to the **[FiveBoosts.net](https://fiveboosts.net)** load simulation platform.

> ⚠️ **Note:** To use this SDK, you need an active subscription and an RSA Server Token from the [FiveBoosts Dashboard](https://fiveboosts.net/dashboard).

---

## 🚀 Key Features

* **Instant Scale Hooks:** Trigger server load scaling (`Walk`, `Drive`, `Fly`) via code based on live events.
* **Emergency Automated Kill-Switch:** Automatically trigger an emergency stop if your server FPS drops below a specific threshold.
* **Live Telemetry Stream:** Fetch live server FPS, RAM usage, and active simulated unit distribution natively.
